# brainfuck-d

## Building

To build use dub:
```
dub build
./brainfuck-d source.b
```

## Example programs

### Hello world

```
$ ./brainfuck-d hello-world.b
Hello World!
```

### Rot13

```
$ ./brainfuck-d rot13.b
ABCD
NOPQ
```
